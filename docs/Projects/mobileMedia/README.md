## 项目--移动多媒体
> 2020-5-13 20:03:23 
<br>分类专栏：vue app

项目简介： 
1. 集电视剧集、音乐播放、书籍浏览和智能机器人闲聊功能的移动端APP
2. 采用vue-cli脚手架搭建，并结合vue-router、vuex
3. 异步请求服务器代理axios和get来抓取豆瓣及网易云接口获取数据，实现功能

项目收获： 对vue-router及vuex的用法加深了理解，使用第三方代理等方法解决基本的跨域问题
